# 侧边栏切换按钮优化 — 设计说明

## 概述

将侧边栏展开/折叠切换按钮从底部移到顶部，用 SVG 双箭头图标替换 `«`/`»` 文本，极简模式下切换按钮作为图标列表第一项，切换时图标有旋转动画。

## 交互行为

- **全宽模式**：切换按钮在侧边栏顶部、el-menu 上方，显示双左箭头（collapse），点击收起侧边栏
- **极简模式**：切换按钮作为图标列表的第一个图标，显示双右箭头（expand），点击展开侧边栏
- **动画**：切换时图标容器有 `transform: rotate(180deg)` 过渡动画，时长 0.25s

## 布局

```
全宽模式 (240px)              极简模式 (60px)
┌────────────────────┐        ┌──────┐
│  ◀◀               │        │  ▶▶  │  ← 第一个图标即切换按钮
├────────────────────┤        │  🏠  │
│  🏠  首页          │        │  📊  │
│  📊  销售管理       │        │  ... │
│  ...               │        └──────┘
└────────────────────┘
```

## 技术方案

- 新增两个 SVG 图标 `collapse` 和 `expand` 到 `icons.js`
- 全宽模式：在 el-menu 上方渲染切换按钮，使用 `collapse` 图标
- 极简模式：在 `sidebar-icons` 列表最前面追加切换按钮，使用 `expand` 图标
- CSS `transition` 实现图标旋转动画
- 删除底部旧切换按钮及 border-top 样式
