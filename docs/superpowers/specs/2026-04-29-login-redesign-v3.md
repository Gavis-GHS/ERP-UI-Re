# 登录页优化设计 v3

## 概述

在 v2（7:3 轮播+登录）基础上优化：增大卡片、丰富右侧内容、Blob 限定右半区域游动且颜色更浓。

## 修改文件

### `src/pages/LoginPage.vue`

脚本逻辑不变，仅修改 template（右侧卡片内容）和 style（卡片尺寸、blob 位置和颜色）。

## 右侧卡片内容结构

从上到下：

| 元素 | 规格 |
|------|------|
| 系统名称 "思嘉供应商平台" | 12px, `#86868b`, 居中 |
| Logo | 120px, 居中, `margin-bottom: 16px` |
| 主标题 "欢迎回来" | 20px, `#1d1d1f`, font-weight 600, 居中 |
| 副标题 "登录您的账户" | 13px, `#86868b`, 居中 |
| 分隔线 | 1px, `#f0f0f0`, `margin: 20px 0` |
| 用户名标签 + 输入框 | 保持现有规格 |
| 密码标签 + 输入框 | 保持现有规格 |
| 登录按钮 | 保持现有规格 |

## 卡片规格

| 属性 | 旧值 | 新值 |
|------|------|------|
| 宽度 | 340px | 380px |
| 内边距 | 40px 36px | 48px 40px |

## Blob 调整

### 位置限定右半区域

所有 blob 的 `left` 从 `50%`~`100%`（右半区域），不再出现在左侧轮播区。

| Blob | 新位置 | 新大小 |
|------|--------|--------|
| blob-blue | top: 5%, right: 10% | 350x350px |
| blob-green | top: 55%, right: 20% | 260x260px |
| blob-purple | top: 25%, right: 5% | 220x220px |
| blob-amber | bottom: 5%, right: 35% | 180x180px |

### 颜色更浓

| Blob | 原中心透明度 | 新中心透明度 | 新渐变过渡点透明度 |
|------|-------------|-------------|-------------------|
| 蓝  | 0.45 | 0.65 | 0.20 |
| 绿  | 0.35 | 0.55 | 0.15 |
| 紫  | 0.30 | 0.50 | 0.12 |
| 橙  | 0.25 | 0.45 | 0.10 |

### 动画

位移范围和速度保持不变，仅起始位置移至右半区域。

## 左侧轮播区

不变（flex: 7，Swiper fade，8 张图，4s 间隔）。

## 背景

blob 层 + 毛玻璃遮罩不变，仅 blob 位置移至右半区域。

## 交互

不变（loading 禁用按钮、回车触发、输入验证、ElMessage 提示）。
