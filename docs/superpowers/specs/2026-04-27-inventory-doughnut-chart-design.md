# 首页库存金额分布环形图卡片 — 设计说明

## 概述

在首页（HomePage.vue）新增一张「库存金额分布」环形图卡片，展示成品库存按物料类型的金额分布，含交互式动态效果。

## 布局定位

首页卡片 grid 为 2 列，现有 5 张卡片。新卡片插入位置：第三行左列（采购概览下方、销售额排名左侧）。

```
年销售情况（全宽折线图）
采购概览（KPI网格）  |  新增客户情况（KPI网格）
月度产值（柱状图）    |  库存金额分布 ★新（环形图）
销售额排名（横向柱状图）
```

## 卡片结构

```
┌──────────────────────────────────┐
│  库存金额分布                      │
├──────────────────────────────────┤
│  总库存金额        环比变化         │
│  ¥12,680万        ↓3.2%（绿色）    │
│                                  │
│   ┌──────────┐    ● 气罐材料 45.2% │
│   │  环形图    │    ● 空间布材料    │
│   │ 中心文字   │    ● PVC膜        │
│   │           │    ● 网格材料      │
│   └──────────┘    ● 水膜材料      │
└──────────────────────────────────┘
```

## 动态交互

| 交互     | 行为                                       |
| -------- | ------------------------------------------ |
| 入场动画 | 页面加载后环形图从 0 旋转展开，约 800ms     |
| 悬停扇区 | 扇区外扩 8px，中心文字切换为当前物料名+金额 |
| 移开鼠标 | 扇区复位，中心文字恢复为默认物料            |
| 点击图例 | 切换对应扇区显示/隐藏，图例项变灰半透明     |

## 数据层

`src/data/dashboard.js` — 新增 `inventoryStats` 字段：

- `totalAmount` — 库存总金额
- `momChange` — 环比变化百分比
- `categories[]` — 物料分布数组，每项含 name / value / color

## 实现要点

- **图表库**：Chart.js doughnut，与首页现有图表统一
- **中心文字**：Chart.js plugin，`afterDraw` 钩子中用 Canvas 绘制居中文字
- **悬停文字切换**：plugin 中监听 `mousemove`，依据 Chart.js 活跃元素更新中心文字
- **入场动画**：`requestAnimationFrame` 逐帧递增数据值至目标
- **图例点击**：维护隐藏集合，click 事件更新 dataset hidden 状态后 `chart.update()`
- **生命周期**：`onBeforeUnmount` 中 `chart.destroy()`，与现有图表一致

## 不涉及

- 不引入新依赖（Chart.js 已存在）
- 不修改路由或全局布局
- 不改动现有卡片
