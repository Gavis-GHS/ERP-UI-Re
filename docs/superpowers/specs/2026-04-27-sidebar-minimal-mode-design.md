# 侧边栏极简模式 — 设计说明

## 概述

侧边导航栏支持在「全宽模式」和「极简模式」之间切换。极简模式下仅显示一级菜单图标（~60px 宽），鼠标悬停有子菜单的模块时弹出浮动面板显示子菜单项。

## 交互行为

- **模式切换**：侧边栏底部折叠按钮（`«` / `»`），点击在两种模式间切换
- **极简模式 — 有子菜单项**：悬停时右侧弹出浮动面板，面板内递归渲染完整子菜单树
- **极简模式 — 无子菜单项**：点击直接导航跳转，不弹窗
- **极简模式 — 弹窗关闭**：鼠标离开弹出面板后自动消失
- **当前激活项高亮**：两种模式下均保持当前路由高亮

## 技术方案

- **状态管理**：`App.vue` 中 `isSidebarCollapsed` ref，通过 prop 传给 AppSidebar
- **全宽模式**：保持现有 el-menu 渲染不变
- **极简模式**：纯 div 列表渲染一级图标 + 绝对定位弹出面板。弹出面板数据复用 MenuData 递归渲染
- **不引入新依赖**：全部用 Vue 3 + Element Plus 现有 API 实现

## 涉及文件

| 文件 | 改动 |
|------|------|
| `src/App.vue` | 新增 `isSidebarCollapsed` ref，传给 AppSidebar |
| `src/components/AppSidebar.vue` | 新增 `collapsed` prop；新增极简模式模板（图标列表 + 弹出面板）；切换按钮；作用域样式 |
| `src/style.css` | 极简模式过渡动画（可选） |

## 不涉及

- 不改动 TopNav
